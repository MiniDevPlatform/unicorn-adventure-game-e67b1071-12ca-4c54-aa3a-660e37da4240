# Unicorn Adventure Game

Create a Unicorn Adventure Game with fairies, a pink princess and magical world.

## Quick Start

```bash
npm install
npm run dev
```

## ONE Template

This project uses the **MiniDev ONE Template** - one template, infinite possibilities.

All configuration in `src/lib/config.ts`.

### Configuration
- **Type**: game
- **Category**: game
- **Difficulty**: medium
- **Players**: solo
